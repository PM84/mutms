# My certifications block for Moodle™ LMS

![Moodle Plugin CI](https://github.com/mutms/moodle-block_mucertify_my/actions/workflows/moodle-ci.yml/badge.svg)

To use this block you need to install [Certifications plugin for Moodle™ LMS](https://github.com/mutms/moodle-tool_mucertify).

See [online documentation](https://github.com/mutms/moodle-tool_mucertify/wiki) for more information.
