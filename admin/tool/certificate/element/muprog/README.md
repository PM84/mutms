# Program fields plugin for Certificate plugin for Moodle™ LMS

[![MDL Shield](https://img.shields.io/endpoint?url=https%3A%2F%2Fmdlshield.com%2Fapi%2Fbadge%2Fcertificateelement_muprog)](https://mdlshield.com/plugins/certificateelement_muprog) ![Moodle Plugin CI](https://github.com/mutms/moodle-certificateelement_muprog/actions/workflows/moodle-ci.yml/badge.svg)

Adds program fields to the [Certificate plugin](https://github.com/moodleworkplace/moodle-tool_certificate)
— fully open source under GPL 3.0, with no restrictions on commercial use. Part of the [MuTMS suite](https://github.com/mutms).

Allows certificate templates to include program-specific data such as program name, completion
date, and status.

## Requirements

> This plugin is included in the [MuTMS distribution](https://github.com/mutms/mutms) —
> no manual installation needed if you use the distribution.

This plugin is recommended for the [Programs plugin](https://github.com/mutms/moodle-tool_muprog).

## Documentation

See [online documentation](https://docs.mutms.org/muprog/) for more information.
