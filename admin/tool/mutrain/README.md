# Training plugin for Moodle™

Training plugin is designed to allow teachers to assign training values
to courses and other entities via Training value custom fields.

Programs completion may be set up to depend on training points users obtain
by completing courses and other activities.

To use this plugin need to install [Programs for Moodle™](https://github.com/mutms/moodle-tool_muprog) 
and [Training value custom field for Moodle™](https://github.com/mutms/moodle-customfield_mutrain) plugins.
